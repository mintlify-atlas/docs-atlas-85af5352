---
title: Badge
description: API reference for the Badge component used to display small labels and tags.
---

The `<Badge>` component displays small labels or tags inline with text. It supports multiple color variants and sizes.

## Import

```tsx
import { Badge } from '@astrojs/starlight/components';
```

## Props

<ParamField path="text" type="string" required>
  The text content to display inside the badge.
</ParamField>

<ParamField path="variant" type="'note' | 'danger' | 'success' | 'caution' | 'tip' | 'default'" default="'default'">
  The color variant of the badge.
  
  - `default` - Accent color theme
  - `note` - Blue theme
  - `danger` - Red theme
  - `success` - Green theme
  - `caution` - Orange theme
  - `tip` - Purple theme
</ParamField>

<ParamField path="size" type="'small' | 'medium' | 'large'" default="'small'">
  The size of the badge.
</ParamField>

<ParamField path="class" type="string">
  Optional CSS class name(s) to apply to the badge element.
</ParamField>

## Additional Props

The Badge component also accepts all standard HTML `<span>` element attributes.

## Usage

### Basic badge

```mdx
<Badge text="New" />
```

### Different variants

```mdx
<Badge text="Note" variant="note" />
<Badge text="Deprecated" variant="danger" />
<Badge text="Stable" variant="success" />
<Badge text="Warning" variant="caution" />
<Badge text="Tip" variant="tip" />
```

### Different sizes

```mdx
<Badge text="Small" size="small" />
<Badge text="Medium" size="medium" />
<Badge text="Large" size="large" />
```

### In headings

```mdx
## My Feature <Badge text="Beta" variant="caution" />
```

### With custom class

```mdx
<Badge text="Custom" class="my-custom-badge" />
```
