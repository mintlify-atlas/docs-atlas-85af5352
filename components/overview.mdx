Starlight provides a comprehensive set of built-in components for common documentation use cases. These components are available from the `@astrojs/starlight/components` package and can be used in both MDX and Markdoc files.

## Importing Components

### In MDX Files

Import components from `@astrojs/starlight/components` in your MDX files:

```mdx

<Aside type="tip">
  This is a helpful tip!
</Aside>
```

### In Markdoc Files

With the Starlight Markdoc preset, components are available using the `{% %}` tag syntax without imports:

```markdoc
{% aside type="tip" %}
This is a helpful tip!
{% /aside %}
```

## Component Categories

### Content Organization

- **Cards & Card Grids** - Group and display related content
- **Tabs** - Present alternative options or code examples
- **Steps** - Guide users through sequential processes
- **FileTree** - Show project structure and file organization

### Information Highlighting

- **Asides** - Draw attention to important information
- **Badges** - Label and categorize content
- **Code** - Display syntax-highlighted code snippets

### Navigation & Actions

- **LinkButton** - Create prominent call-to-action links
- **LinkCard** - Display navigational links as cards

### Visual Elements

- **Icons** - Add visual indicators and decorative elements
