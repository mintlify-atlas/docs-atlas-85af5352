Starlight is a full-featured documentation theme built on top of the [Astro](https://astro.build) framework. Created to help you build beautiful, accessible, and high-performance documentation sites with minimal configuration.

  <Card title="Lightning Fast" icon="bolt">

  </Card>

  </Card>

</CardGroup>

## Why Starlight?

Starlight combines the best of modern web development with the specific needs of documentation sites:

### Performance First

Starlight generates static HTML at build time, resulting in blazing-fast page loads. With intelligent caching strategies and minimal JavaScript, your documentation loads instantly and stays responsive.

<Note>
  According to the Website Carbon Calculator, Starlight produces only **0.01g of CO₂ per page visit**, making it cleaner than 98% of web pages tested.
</Note>

### Developer Experience

Built on Astro, Starlight offers a familiar development experience:

- **File-based routing** — Create pages by adding Markdown, MDX, or Markdoc files
- **Component-driven** — Use Astro components or bring your own React, Vue, or Svelte components
- **TypeScript support** — Full type safety for configuration and frontmatter
- **Hot module replacement** — See changes instantly during development

### Production Ready Features

Starlight includes everything you need for professional documentation:

<Steps>
  <Step title="Built-in Components">
    Cards, tabs, code blocks with syntax highlighting, file trees, and more — all accessible and mobile-friendly.
  </Step>
  
  <Step title="Automatic Navigation">
    Sidebar navigation is automatically generated from your file structure, with support for manual ordering and grouping.
  </Step>
  
  <Step title="Table of Contents">
    Auto-generated table of contents for each page makes it easy to navigate long-form documentation.
  </Step>
  
  <Step title="Edit on GitHub Links">
    Encourage contributions with automatic "Edit this page" links to your repository.
  </Step>
</Steps>

## Quick Start

Get started with Starlight in minutes:

```bash
npm create astro@latest -- --template starlight
```

This creates a new Astro project with Starlight pre-configured and ready to go. Check out the [Quickstart guide](/quickstart) for detailed instructions.

## Use Cases

Starlight is perfect for:

- **Product Documentation** — Document your SaaS product, API, or SDK
- **Technical Guides** — Create comprehensive tutorials and how-to guides  
- **Component Libraries** — Showcase your design system or component library
- **Knowledge Bases** — Build internal wikis and knowledge repositories
- **Open Source Projects** — Provide clear documentation for your open source tools
