## Share Your Plugin

Built a plugin or tool for Starlight? Share it with the community:

1. Add your plugin to the [Plugins & Integrations page](https://github.com/withastro/starlight/blob/main/docs/src/content/docs/resources/plugins.mdx)
2. Open a pull request with a brief description
3. Join the discussion in the [#starlight channel](https://astro.build/chat) on Discord

To learn how to build a Starlight plugin, check out the [Plugins Reference](/reference/plugins) documentation.
