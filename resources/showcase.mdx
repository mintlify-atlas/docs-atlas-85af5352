## Add Your Site

Have you built a documentation site with Starlight? You can add it to the official showcase:

1. Fork the [Starlight repository](https://github.com/withastro/starlight)
2. Add a screenshot (800×450px PNG) to `docs/src/assets/showcase/`
3. Add your site entry to `docs/src/components/showcase-sites.astro`
4. Open a pull request

See the full instructions in the [Contributing Guide](https://github.com/withastro/starlight/blob/main/CONTRIBUTING.md#showcase).
